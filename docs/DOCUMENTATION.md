# Документация проекта

## Общее описание решения
Проект предоставляет модули для вычисления площадей и периметров геометрических фигур: круга, прямоугольника, квадрата и треугольника.

## Описание функций

### Модуль circle.py
- area(r)  
  Вычисляет площадь круга.

  принимает радиус r и возвращает площадь, равной pi * r * r
    параметры:
    r (float): радиус круга
    возвращаемое значение: area(float): площадь круга

  Пример:  
  area(2)  # 28.274333882308138

- **perimeter(r)**
  Вычисляет длину окружности.

  принимает радиус r и возвращает периметр, равный 2 * pi * r
    параметры:
    r (float): радиус круга
    возвращаемое значение: perimeter(float): площадь круга
    
  Пример:
  perimeter(3)  # 18.84955592153876

### Модуль rectangle.py
- area(a, b)
  Вычисляет площадь прямоугольника.

  принимает ширину a и высоту b и возвращает площадь, равной произведению ширины a и высоты b
    параметры:
    a (float): ширина прямоугольника
    b (float): высота прямоугольника
    возвращаемое значение: area(float): площадь прямоугольника
    
  Пример:
  area(4, 5)  # 20

- **perimeter(a, b)**
  Вычисляет периметр прямоугольника.

  принимает ширину a и высоту b и возвращает периметр, равный (a + b) * 2
    параметры:
    a (float): ширина прямоугольника
    b (float): высота прямоугольника
    возвращаемое значение: perimeter(float): периметр прямоугольника

  Пример:
  perimeter(3, 4)  # 14

### Модуль square.py
- area(a)
  Вычисляет площадь квадрата.

  принимает сторону квадрата a и возвращает площадь квадрата a * a
    параметры:
    a (float): сторона квадрата
    возвращаемое значение: area(float): площадь квадрата 

  Пример:
  area(5)  # 25

- **perimeter(a)**
  Вычисляет периметр квадрата.

  принимает число a и возвращает периметр равный произведеню 4 * a
    параметры:
    a (float): сторона квадрата
    возвращаемое значение: perimetr(float): периметр квадрата

  Пример:
  perimeter(4)  # 16

### Модуль triangle.py
- area(a, h)
  Вычисляет площадь треугольника.

  принимает основание a и высоту h, опущенную на основание a. возвращает площадь, равной a * h * 0.5
    параметры:
    a (float): основание треугольника
    h (float): высота треугольника, опущенная на основание треугольника
    возвращаемое значение: area(float): площадь треугольника

  Пример:
  area(6, 6)  # 18

- **perimeter(a, b, c)**
  Вычисляет периметр треугольника.

  принимает стороны треугольника a, b, c и возвращает периметр, равный сумме a, b, c
    параметры:
    a (float): первая сторона треугольника
    b (float): вторая сторона треугольника
    c (float): третья сторона треугольника
    возвращаемое значение: area(float): периметр треугольника

  Пример:
  perimeter(1, 2, 3)  # 6
